## Overview

TabFlip is a browser extension that helps users switch between recently used tabs using a keyboard shortcut. we take your privacy seriously. this policy explains what data TabFlip accesses and how it is handled.

## Data Collection

TabFlip does not collect, store, transmit, or share any personal data. period.

## What TabFlip Accesses Locally

TabFlip accesses the following browser data to function. all of this stays on your device and is never sent anywhere:

- **Tab metadata:** tab titles, URLs, and favicon URLs are read to display the tab switcher overlay.
- **Tab screenshots:** screenshots of your active tabs are captured to show page previews in the switcher. these are cached temporarily in browser memory and are never saved to disk or transmitted.
- **Tab activity order:** TabFlip tracks which tabs you visited most recently (up to 5) to display them in order. this is stored in memory only and resets when the browser is closed.

## Data Storage

All data used by TabFlip is stored in local browser memory only. nothing is written to disk, uploaded to any server, or shared with any third party.

## Third Party Services

TabFlip does not use any third party analytics, tracking, advertising, or data collection services.

## Remote Code

TabFlip does not load or execute any remote code. all functionality is bundled within the extension package.

## Permissions

TabFlip requests the following browser permissions:

- **tabs:** to read tab titles, URLs, and favicons for the switcher UI
- **activeTab:** to inject the switcher overlay on the current page
- **scripting:** to programmatically inject the overlay when the shortcut is triggered
- **host permissions:** to capture tab screenshots for page previews
