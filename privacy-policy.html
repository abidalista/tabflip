TabFlip — Privacy Policy

Effective date: March 1, 2026

What TabFlip does

TabFlip is a browser tab switcher. It tracks which tabs you've used most recently so you can quickly switch between them using a keyboard shortcut.

Data collected

TabFlip does not collect, transmit, or store any personal data. Specifically:

  • Tab metadata (titles, URLs, favicons) is read only to display the tab switcher overlay. This data stays entirely in your browser's local memory and is never sent anywhere.
  • Tab screenshots are captured temporarily in memory to show visual previews. They are never saved to disk, never uploaded, and are discarded when the extension's service worker resets.
  • No analytics, telemetry, or tracking of any kind is included.
  • No network requests are made by this extension. It operates entirely offline.

Permissions explained

  • tabs — Read tab titles and URLs to display in the switcher.
  • activeTab — Access the currently active tab for screenshot capture.
  • scripting — Inject the overlay UI into web pages.
  • <all_urls> — Required for the content script (keyboard listener + overlay) to run on all pages, and for captureVisibleTab to capture screenshots on any tab.

Third parties

TabFlip shares no data with any third party. There are no ads, no analytics services, and no external dependencies.
